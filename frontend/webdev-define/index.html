What is WebDev

Basically building the website for the internet.

  1. html : provides the structure of the website
  2. css: css gives the style to the structure
  3. js: js gives the functionality to the website

when accessing the website it follows request and response cycle.

client browser sent the reques to the server for example amazon.com then amazon.com server sends the data in the form of the code then the our client browser render the code and show the data in the website design format so this process continues. this is called request response cycle.
