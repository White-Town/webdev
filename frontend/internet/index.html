what is internet??

client vs server

client is the computer devices or mobile or any electronic device that request for the data to the server

server is the one who serve the data to the client and it is in online 24/7

  • ip adddress- internet protocol address. unique address assigned to each device that is connected to the internet. that is bascially the address of the each device connected to the internet.
  • ISP: Internet Serivce Provider. these are the company that provide the ip address to the each device connected to the inernet and also provide the internet connection. like worldlink, jio, ntc
  • DNS: Domain Name System. These are the system server that converts domain name to the ip address so that the device can connect to the internet
